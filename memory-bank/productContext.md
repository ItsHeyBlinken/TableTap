# Product Context

**TableTap** (UI) / **ShowPOS** (internal) — sports card **vendor POS + profit tracker** for show vendors, flippers, and table sellers. Not a collector portfolio app.

Primary question: *What did I sell today, and how much did I actually make?*

- Sales-first: fast record-sale flow, today’s profit on dashboard
- Stock supports selling (cost basis, search, sell from list)
- Sales events group show/weekend/stream profit
- Trades: card out + card in + optional cash adjustment
- **Mobile-first** at the table (bottom nav, large inputs, card lists not wide tables)

Target environment: phone at card shows; laptop optional.
